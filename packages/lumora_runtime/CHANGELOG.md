# Changelog

All notable changes to the Lumora Runtime package will be documented in this file.

## [0.1.0] - 2025-01-11

### Added
- Initial package structure for Lumora Runtime
- Core interpreter for Lumora IR schemas
- Widget registry system for extensible rendering
- State management with StateManager
- Event bridge for user interactions
- Prop parsing utilities (padding, color, text style, alignment)
- Key generation for stable widget identity
- Error handling with fallback widgets
- Public API exports

### Features
- Dynamic widget interpretation from JSON schemas
- Support for state references ($variableName)
- Support for event handlers
- Recursive widget tree building
- Graceful error handling with visual feedback

### Documentation
- README with usage examples
- API documentation in code
- Architecture overview
